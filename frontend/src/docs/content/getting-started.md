# LightBridge 文档中心

欢迎来到 LightBridge 文档中心。这里会逐步沉淀项目各个模块与功能的使用说明，包含控制台操作、管理员配置、渠道管理、支付订阅、模块市场等内容。

## 文档维护方式

文档使用 Markdown 编写，支持常见语法：标题、段落、列表、表格、代码块、引用和图片。

```markdown
![图片说明](./images/example.png)
```

后续新增文档时，可以在 `frontend/src/docs/content/` 中添加 Markdown 文件，并在 `frontend/src/docs/index.ts` 中注册它。

## 建议结构

- 用户控制台：API Key、用量查询、充值/订阅、个人资料。
- 管理控制台：用户、分组、渠道、账号、系统设置。
- 运维与风控：运维监控、错误分析、隐私过滤、风控中心。
- 扩展能力：模块市场、自定义页面、外部集成。

## 图片支持

Markdown 中可以直接插入远程图片，也可以使用前端资源路径。图片会在阅读区内自适应宽度展示。

## 下一步

你可以从这篇入口文档开始，把 LightBridge 的模块说明逐步拆成多篇文档，左侧列表会负责导航，右侧目录会自动从正文标题生成。
