# 管理员 - 渠道与账号

本章介绍渠道定价、渠道监控、账号管理和代理管理功能。

---

## 渠道定价管理 `/admin/channels/pricing`

管理 API 渠道的定价规则、模型映射和平台配置。

### 列表操作

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 搜索渠道 | 输入搜索关键词 | 按名称搜索 |
| 按状态筛选 | 选择状态下拉 | All / Active / Disabled |
| 创建渠道 | 点击「创建渠道」按钮 | 打开多标签创建表单 |
| 编辑渠道 | 点击编辑按钮 | 打开编辑表单 |
| 切换状态 | 点击状态切换开关 | 内联切换 active / disabled |
| 删除渠道 | 点击删除按钮 | 二次确认后删除 |

### 创建/编辑表单

**基础设置 Tab**：

| 字段 | 说明 |
|------|------|
| 名称 | 渠道名称（必填） |
| 描述 | 渠道描述 |
| 限制模型 | 仅允许定价列表中的模型 |
| 计费基础 | 按渠道映射模型 / 按请求模型 / 按最终上游模型 |
| 平台配置 | 选择 Anthropic / OpenAI / Gemini / Grok / Antigravity |
| 应用定价到账号统计 | 开关 |

**平台 Tab（Anthropic / OpenAI / Gemini / Grok / Antigravity 各一个）**：

| 字段 | 说明 |
|------|------|
| 关联分组 | 复选框列表，含冲突检测 |
| Web 搜索模拟 | 仅 Anthropic |
| Codex 图片生成桥接 | 仅 OpenAI |
| Bedrock CC 兼容 | 仅 Anthropic |
| 模型映射 | 动态键值对（源模型 → 目标模型） |
| 模型定价 | 多维度定价：输入/输出/缓存写入/缓存读取/图片输出/按请求 |
| 账号统计定价规则 | 按规则组/账号设置独立定价 |

## 渠道监控 `/admin/channels/monitor`

创建和管理渠道健康监控器。

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 搜索监控器 | 输入搜索关键词 | 按名称搜索 |
| 按提供商筛选 | 选择提供商下拉 | 按上游提供商过滤 |
| 创建监控器 | 点击「创建监控器」按钮 | 打开创建表单 |
| 编辑监控器 | 点击编辑按钮 | 打开编辑表单 |
| 立即运行 | 点击「立即运行」按钮 | 触发健康检查并显示结果 |
| 启用/禁用 | 点击状态切换开关 | 内联切换 |
| 删除监控器 | 点击删除按钮 | 二次确认后删除 |
| 管理模板 | 点击「管理模板」按钮 | 打开模板管理弹窗 |

监控器显示：名称、提供商、主要模型、7 天可用性百分比、主要延迟（ms）、启用状态。

## 账号管理 `/admin/accounts`

上游 API 账号的全生命周期管理。

### 搜索与筛选

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 搜索账号 | 输入搜索关键词 | 模糊搜索 |
| 按平台筛选 | 选择平台下拉 | 按上游平台过滤 |
| 按类型筛选 | 选择类型下拉 | 按账号类型过滤 |
| 按状态筛选 | 选择状态下拉 | active / rate_limited / temp_unschedulable / unschedulable / disabled |
| 按分组筛选 | 选择分组下拉 | 包含"未分组"选项 |
| 按隐私模式筛选 | 选择隐私下拉 | 按隐私模式过滤 |
| 自动刷新 | 点击自动刷新下拉 | 可选 5 / 10 / 15 / 30 秒间隔 |

### 账号操作

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 创建账号 | 点击创建按钮 | 打开创建弹窗（含代理和分组分配） |
| 编辑账号 | 点击编辑按钮 | 打开编辑表单 |
| 删除账号 | 点击删除按钮 | 二次确认后删除 |
| 测试账号 | 点击测试按钮 | 打开测试弹窗 |
| 查看统计 | 点击统计按钮 | 打开统计弹窗 |
| 安排测试 | 点击安排测试 | 选择模型后安排定时测试 |
| 重新认证 | 点击重新认证 | 打开重新认证弹窗 |
| 刷新令牌 | 点击刷新令牌 | 调用刷新凭证 API |
| 恢复状态 | 点击恢复状态 | 调用恢复状态 API |
| 验证真伪 | 点击验证真伪 | 验证 Claude 模型真伪 |
| 重置配额 | 点击重置配额 | 重置账号配额 |
| 设置隐私 | 点击设置隐私 | 设置隐私模式 |
| 同步余额 | 点击同步余额 | 同步 NewAPI / LBC 余额 |
| 打开控制台 | 点击打开控制台 | 在新标签页打开上游控制台 |

### 批量操作

选中多行后，底部出现批量操作栏：

| 功能 | 说明 |
|------|------|
| 批量删除 | 删除所有选中账号 |
| 批量重置状态 | 清除错误状态 |
| 批量刷新令牌 | 刷新所有选中账号的令牌 |
| 批量编辑 | 打开批量编辑弹窗 |
| 切换可调度 | 启用/禁用账号调度 |

### 更多操作

| 功能 | 说明 |
|------|------|
| 同步 CRS | 从 CRS 同步账号 |
| 导入数据 | 打开导入弹窗 |
| 导出数据 | 导出为 JSON（可选含代理） |
| 错误透传规则 | 管理错误透传规则 |
| TLS 指纹配置 | 管理 TLS 指纹配置 |
| 列设置 | 切换列的显隐 |

## 代理管理 `/admin/proxies`

代理服务器的管理和质量检测。

### 列表操作

| 功能 | 操作方式 | 说明 |
|------|---------|------|
| 搜索代理 | 输入搜索关键词 | 按名称搜索 |
| 按协议筛选 | 选择协议下拉 | All / HTTP / HTTPS / SOCKS5 / SOCKS5H |
| 按状态筛选 | 选择状态下拉 | All / Active / Inactive |
| 创建代理 | 点击「创建代理」按钮 | 支持标准添加和批量添加 |
| 编辑代理 | 点击编辑按钮 | 打开编辑表单 |
| 删除代理 | 点击删除按钮 | 有账号使用时阻止删除 |
| 测试连接 | 点击测试连接按钮 | 测试延迟、IP、地理位置 |
| 质量检测 | 点击质量检测按钮 | 多步骤质量评估（OpenAI / Anthropic / Gemini） |
| 复制代理 URL | 点击复制按钮 | 支持多种格式 |
| 查看关联账号 | 点击账号数量 | 打开账号列表弹窗 |
| 导入/导出数据 | 点击对应按钮 | JSON 格式导入导出 |

### 创建代理模式

**标准添加**：填写名称、协议（HTTP / HTTPS / SOCKS5 / SOCKS5H）、主机、端口、用户名（可选）、密码（可选）。

**批量添加**：粘贴代理 URL 列表（`protocol://user:pass@host:port` 格式），实时解析显示有效/无效/重复数量。

### 质量报告

- 总体评分和等级
- 出口 IP、国家、基础延迟
- 各目标（OpenAI / Anthropic / Gemini）的详细检测结果
