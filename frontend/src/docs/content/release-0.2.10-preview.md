# 0.2.10-preview 版本更新

## 修复问题

1. 修复 0.2.9-preview 已安装环境无法继续升级的问题
   - 更新器现在会识别 `-preview.1`、`-rc.2` 等预发布递增版本，不再只比较 `MAJOR.MINOR.PATCH`。
   - 补发为 `0.2.10-preview`，确保已安装 `0.2.9-preview` 的实例也能通过自动升级检测到新版本。

2. 继续包含 0.2.9-preview.1 的分组调度修复
   - 分组内账号不再受分组所属平台限制，组内混合协议账号均可进入轮询候选池。
   - 修复教研平台等非同平台分组中仍可能返回 `503 no available accounts` 的问题。

## 优化调整

1. 预览版更新路径更稳健
   - 预览版仍优先安装对应平台的增量二进制包。
   - 后续 preview 补丁版本可以正常被识别为可升级版本。

## 验证与发布备注

1. 已补充更新器回归验证
   - 覆盖 `0.2.9-preview` 到 `0.2.9-preview.1` 的预发布递增检测。
   - 覆盖 preview、rc、正式版与 build metadata 的版本比较行为。
