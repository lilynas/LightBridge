# 更新日志

## v0.2.26

- 模型列表支持监控状态查看与一键配置，侧边栏菜单分组整理，升级弹窗支持直接重启，版本号显示优化

### 详细内容

#### 模型列表集成监控状态
- 每个模型卡片可直接查看监控状态、7 天可用率和响应延迟
- 管理员可从模型列表一键为任意模型配置监控，复用现有的监控配置表单
- 选择账号来源后自动填充模型名称和提供商信息

#### 模型列表分组筛选
- 按分组视图下新增分组筛选器，可快速筛选特定分组的模型
- 分组筛选器支持下拉选择，显示每个分组的模型数量
- 分组数量过多时支持滚动浏览

#### 侧边栏菜单分组
- 管理后台菜单按功能模块分为七个分组，用户端菜单分为三个分组
- 每个分组支持折叠展开，收起状态下点击图标可直接进入对应功能

#### 升级弹窗增加重启按钮
- 升级内容弹窗新增"立即重启"按钮，升级完成后可直接重启服务

#### 版本号显示优化
- 侧边栏版本号精简显示，Preview 版本通过颜色区分

#### Bug 修复
- 修复模型列表页面可能出现的空白问题

#### 样式调整
- 模型列表来源数量徽章改为红色，与其他页面风格统一

#### 表格排序功能增强
- 账户管理、分组管理、渠道管理等表格支持更多列的排序

#### 营销功能渐进式结构
- 公告、邀请记录、返现记录、提取记录、兑换码、优惠码等营销推广功能支持独立开关
- 可通过系统设置按需启用/禁用各功能模块，节省性能开销
- 新增 feature flags：announcements_enabled、redeem_enabled、promo_enabled

#### 渠道定价与 IP 管理渐进式结构
- 渠道定价功能支持独立开关，禁用时不加载相关组件
- IP 管理功能支持独立开关，禁用时不加载相关组件
- 新增 feature flags：channel_pricing_enabled、proxies_enabled
- 后端 ChannelMonitorRunner 在功能禁用时不启动，实现真正的渐进式加载

#### 系统设置界面优化
- 设置页面顶部 Tab 导航改为底部边框样式，与整体界面更协调
- 邮件验证功能从系统设置抽离，移至模块中心统一管理
- 模块中心新增邮件验证功能卡片，支持一键启用/禁用和配置跳转

#### 渐进式组件规范
- 创建标准渐进式组件规范文档
- 登录条款功能实现为渐进式组件，支持独立启用/禁用
- 模块中心新增登录条款功能卡片
